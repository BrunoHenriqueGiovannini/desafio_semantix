#######################################################################################################
Para executar o programa é necessário ter a linguagem R instalada na máquina e o RStudio instalado.
#######################################################################################################

Segue abaixo os passos para execução:

1 - Criar diretório e salvar os arquivos 'edges.dat' e 'Desafio_Semantix.R' 
2 - Abrir o RStudio e carregar o arquivo 'Desafio_Semantix.R'  
3 - instalar o pacote "igraph"
3 - Carregar o pacote "igraph"
4 - Setar o diretório no RStudio para carregar o dataset 'edges.dat'
5 - Executar os comandos linha a linha (CTRL + Enter)
